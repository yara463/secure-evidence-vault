How the System Works

The project protects evidence in two important ways: confidentiality and integrity.

1
Upload the original evidence file
2
Generate a unique SHA-256 digital fingerprint
3
Encrypt the file before storing it
4
Later verify if the evidence is still unchanged

Security Goals

🔒 Confidentiality

The file is encrypted, so it cannot be read directly.

🧬 Integrity

SHA-256 makes a unique fingerprint for the file.

🚨 Tampering Detection

If the file changes, the hash changes, and the system detects it.
